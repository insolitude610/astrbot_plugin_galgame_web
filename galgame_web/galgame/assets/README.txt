将角色立绘 PNG 文件放入此目录。
插件会自动检测文件并匹配到对应的情绪/图层：

  表情立绘（single 模式可选，layered 模式头部层）：
    neutral.png  happy.png  sad.png  angry.png
    surprised.png  blush.png  thinking.png

  分层立绘（layered 模式）：
    body.png  hair_back.png  head.png  hair_front.png
    mouth_open.png  mouth_closed.png  orb.png

  场景背景：
    background.png 或 bg.png

如果配置中留空，插件将按上述命名自动匹配。
如果手动填写了文件名，则优先使用手动配置。
